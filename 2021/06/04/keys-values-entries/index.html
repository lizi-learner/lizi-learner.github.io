keys(),values(),entries()迭代器对象

发表于 2021-06-04

keys()，values()，entries()能返回一个迭代器对象，方便我们对数据的迭代。

其中，对于普通对象和一些特殊的数据类型，它们的使用方式是不一样的。

obj.keys

obj.keys()调用语法支持以下数据类型：

  • Map
  • Set
  • Array

示例：

Object.keys，values，entries

对于普通对象可以使用以下方法迭代

  • Object.keys(obj)：返回一个包含该对象所有的键的数组。
  • Object.values(obj)：返回一个包含该对象所有的值的数组。
  • Object.entries(obj)：返回一个包含该对象所有 [key, value] 键值对的数组。

可以利用以上方法让对象能够使用数组方法

  • Object.fromEntries(array)：将数组转换为对象

  • 转换对象

    对象缺少数组存在的许多方法，例如 map 和 filter 等。

    如果我们想应用它们，那么我们可以使用 Object.entries，然后使用 Object.fromEntries：

    • 使用 Object.entries(obj) 从 obj 获取由键/值对组成的数组。

    • 对该数组使用数组方法，例如 map。

    • 对结果数组使用 Object.fromEntries(array) 方法，将结果转回成对象。
